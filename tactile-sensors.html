The Problem

We don't have good tactile data because we don't have good cheap sensors. We don't have good cheap sensors because nobody's figured out how to make them yet. And we don't have the models to use the data even if we did.

Sensor Array Prototype
Breaking the loop with cheap, scalable sensors

I'm building tactile sensors cheap enough to actually scale. The development platform is the LeRobot SO-101 — a real robot, not a simulation. The goal is sensors that can be mounted to any humanoid or robot platform without a custom integration project for each one.

The bigger vision: get real tactile data at scale, feed it back into models, and start closing that loop. Humanoids need touch to do useful work in the real world. That can't happen until the sensors are cheap, robust, and everywhere — not sitting in a single university lab.
