Matrix Display

The Matrix Display project is a fun exploration of electronics and visual design. This 10x10 LED matrix was programmed to display animations and messages, showcasing my skills in soldering, circuit design, and programming.

Project Highlights

  • LEDs: 100 RGB LEDs arranged in a 10x10 grid.
  • Controller: Arduino-based driver for animations.
  • Functionality: Displays animations, text scrolling, and interactive light shows.
  • Build Time: 2 weeks
